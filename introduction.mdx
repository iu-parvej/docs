# Portfolio Website

A comprehensive, feature-rich portfolio website with an admin dashboard for complete content management.

## Tech Stack

| Technology | Purpose |
|------------|---------|
| React 18 | Frontend framework |
| Vite 5 | Build tool & dev server |
| Tailwind CSS 3 | Styling |
| Appwrite | Backend, database & storage |
| Lucide React | Icons |
